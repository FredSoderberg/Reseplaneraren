Detailed Description

A generic graph implementation.

A directed graph with support for node- and edge-labels and path-finding.

Run a function with each of the graph's edges as input.

Runs the function with each of the graph's edge labels, as well as the start- and the end-node of the edge as input. The function will be called with arguments in this order: fun(from, to, edge_label).
